# Services — provenance gates & source-discipline engineering

I help teams install the same **fail-closed, abstain-don't-fabricate** provenance gate this
repo is built on. You are buying **engineering and expertise**, scoped honestly — not a
capability guarantee.

> **Honest scope.** The gate *reduces* hallucinated/unsourced output and makes the rest
> auditable; it does **not** eliminate it. On the validated benchmark it cut attribution-
> hallucination by Δ12.5% on an 8B model — meaningful, not magic. Every engagement states
> its measured target and its residual error up front.
