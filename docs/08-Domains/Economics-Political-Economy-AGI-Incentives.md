# Economics & Political Economy with AGI-Incentive Meta Layer

Status: applied as a Level-3 distribution-shift scaffold via `tools/build_econ_political_shift_pack.py`.

Purpose: stress Sophia outside its current humanities/provenance corpus by testing quantitative claims, regime changes, institutional incentives, and AGI deployment incentives.

Why this is a strong OOD domain:

- moves from textual attribution to empirical/quantitative source discipline;
- introduces dynamic facts and data revisions;
- forces causal/incentive reasoning rather than author/title lookup;
- tests source incentives: who benefits if this claim is believed?;
- connects Sophia's epistemic core to civilization-scale truth anchoring.

Generated scaffold:

```bash
python tools/build_econ_political_shift_pack.py
```

Outputs:

```text
private/shift/econ-political-agi-incentive-spec-<date>.json
agi-proof/level3-candidate-benchmark/<date>-local-smoke/economics_political_economy/fact-check-candidate-report.json
```

The private spec is gitignored and must be filled with real sources before a Level-3 run. The candidate report is a deterministic offline fixture and is not Level-3 evidence.

Recommended real source families:

- IMF / World Bank / OECD / FRED for quantitative macro data;
- NBER / academic papers for causal/incentive claims;
- government statistical agencies for primary data;
- company filings / policy documents for AGI lab incentives;
- independent watchdog/think-tank sources for counter-claims.

High-risk rule: economics, policy, and AGI-incentive claims require **3 independent entailing source families** or the gate holds.
