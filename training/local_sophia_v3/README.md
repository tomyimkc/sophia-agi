# local_sophia_v3 — Sophia-Wisdom-4B M2 dataset

Gate-passed, decontaminated, route-first training data for the wisdom-habit behavioural
target. Built by the **teacher → gate → admission** pipeline:

```bash
python3 tools/build_sophia_wisdom_dataset.py --check     # synth + gate + decontam, no writes
python3 tools/build_sophia_wisdom_dataset.py --stats     # full build + mix report
```

Only `manifest.json` is tracked in git — the `sft_*.jsonl`, `mlx/`, `preference_pairs.jsonl`,
and `rejected.jsonl` are **reproducible artifacts** (gitignored), regenerated by the tool above
from `data/` + `moral_corpus/` + the reused human-curated packs.

## What's here (regenerate to materialise)

| File | Contents |
|---|---|
| `manifest.json` | totals, per-family mix vs target, decontam drops, **M2 go/no-go** |
| `sft_<family>.jsonl` | gated SFT rows per task family (`{messages, metadata}`) |
| `mlx/{train,valid}.jsonl` | MLX-LM ChatDataset reshape (95/5 split by prompt hash) |
| `preference_pairs.jsonl` | `{prompt, chosen, rejected}` for ORPO (M4): chosen = gated answer, rejected = fabrication |
| `rejected.jsonl` | audit trail of gate-rejected candidates (negatives, not training gold) |

## Honest bounds

- **Teacher = deterministic templating + human-curated reuse.** The live-LLM teacher path
  (`--teacher`) needs model egress, which the build environment blocks; it is the route to the
  10–20k target. Current yield is **corpus-bounded**, well below 10k.
- **Mix is skewed** toward `source_discipline` and **general-retention is under the 25–30%
  floor** — a real SFT needs a license-clean external instruct slice expanded first, or the
  model becomes a narrow refusal machine. See `manifest.json → go_no_go` and the failure ledger.
- Admission is **real**: every synthetic candidate must pass `agent.gate.check_response`
  (advisor mode) — `tests/test_sophia_wisdom_dataset.py` proves a bare fabrication is rejected.
- Truth lives **outside** the model. These rows teach habits; the external gate enforces truth.
  Not an AGI claim.
